# npm-check-install

## Installation

```sh
$ npm install --save npm-check-install
```

## Usage

```
const npmCheckInstall = require('npm-check-install');

npmCheckInstall('antd', { cwd: 'PATH TO CHECK AND INSTALL' // default process.cwd() });
```
